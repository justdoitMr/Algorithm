## [27. 移除元素](https://leetcode-cn.com/problems/remove-element/)

~~~java
class Solution {
    public int removeElement(int[] nums, int val) {

        return removeElement_A(nums,val);
    }

    public int removeElement_A(int nums[],int val){

        int sz = nums.length;
        int left = 0;
        int right = 0;
        while(right < sz){
            if(nums[right] == val){
                right++;
            }else{
                nums[left] = nums[right];
                left++;
                right++;
            }
        }
        return left;
    }
}
~~~
